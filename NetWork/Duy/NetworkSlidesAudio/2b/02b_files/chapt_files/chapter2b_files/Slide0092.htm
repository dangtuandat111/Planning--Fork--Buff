qClient đọc một dòng kí tự từ input chuẩn (inFromUser stream), gửi tới server qua socket (outToServer)

qserver biến đổi dòng ký tự đó (chữ thường thành chữ hoa) và gửi trả về cho client. qclient đọc dòng ký tự đã biến đổi từ socket, in ra (inFromServer)
